정적 웹서버 : Nginx, Apache 등이 있다. 
서버에 저장된 디렉터리 경로에서 요청한 파일을
찾아 그 내용을 HTTP 응답 바디에 넣어 전송.

동적 웹서버 : 아파치톰캣 등

스프링 부트를 사용할 시
@RestController : JSON을 리턴하는 웹 서비스임을 명시

@GetMapping : Path 설정, GET 메서드 사용

Bean과 Component의 차이점
- Bean은 어떻게 생성해야 하는지, 매개변수를 어떻게 넣어줘야하는지 알려줄 수 있다.
Component는 ComponentScan 어노테이션이 있는 경우 베이스 패키지와 그 하위 패키지에서 Component가 달린 클래스를 찾는다.
Autowired는 Component가 달린 오브젝트의 생성 시 다른 오브젝트에 의존한다면 그 멤버 변수 오브젝트를 찾아 넣어줘야 한다.

gradle 빌드 자동화 도구, 유닛 테스트 실행 가능
-그루비로 코딩됨. 자바를 컴파일하려면 플러그인을 추가해줘야함.

lombok 
- 롬복이 코드를 작성하려면 어노테이션 프로세서라는 것이 필요하기 때문에 configuration부분에 프로세서 작성.

google guava 라이브러리를 이용하여 그래들이 메이븐 리파지토리에 있는 라이브러리들을 자동으로 다운로드 받을 수 있게 했다.

REST 아키텍처 : 클라이언트에게 서비스를 이용하려면 어떤 형식으로 요청하고 응답해야하는지 알려주는 것
위 조건을 따라서 설계 및 구현한 서비스를 RESTful 서비스라고 한다.

레이어드 아키텍처 : 프레젠테이션, 비즈니스, 퍼시스턴스, DB 레이어로 나뉘어 요청과 응답을 받는 구조
